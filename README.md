# CricketAnalytics

## Overview
In this project, I have implemented Machine Learning techniques to predict the effectiveness of a cricket shot played based on bowler, ball type, and other such conditions.

## Approach
Scraped ball-by-ball data from ESPN Cricinfo and developed a novel approach of assigning an effectiveness scale to cricket shots in the analysis parameters based on ball type, outcome, and other contextual conditions. Utilized Sklearn frameworks to implement the predictive model.

## Tech Stack
- Python (Jupyter Notebook)
- scikit-learn
- Web scraping (ESPN Cricinfo ball-by-ball data)

## Files
- `SDP.ipynb` — main analysis and modeling notebook
